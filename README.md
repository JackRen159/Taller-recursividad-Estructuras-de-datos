## Folder Structure

The workspace contains two folders by default, where:

- `src`: the folder to maintain sources
- `lib`: the folder to maintain dependencies

Meanwhile, the compiled output files will be generated in the `bin` folder by default.

> If you want to customize the folder structure, open `.vscode/settings.json` and update the related settings there.

## Dependency Management

The `JAVA PROJECTS` view allows you to manage your dependencies. More details can be found [here](https://github.com/microsoft/vscode-java-dependency#manage-dependencies).

## Descripción del Taller
Este repositorio contiene la solucion a los ejercicios propuestos en el taller de recursividad. El objetivo es aplicar conceptos de logica recursiva para resolver problemas matematicos, manipulación de estructuras y algoritmos avanzados utilizando el lenguaje Java.

El proyecto está dividido en los siguientes modulos de logica:
* **Estructuras recursivas:** Implementación en estructuras de datos.
* **MathRecursivo:** Operaciones matemáticas mediante recursión.
* **NumeroUtils:** Utilidades para procesamiento de números.
* **OperacionesRecursivas:** Algoritmos base.
* **RecursividadAvanzada:** Casos de estudio de mayor complejidad.
